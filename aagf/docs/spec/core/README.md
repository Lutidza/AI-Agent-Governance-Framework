# core

Core-спецификации независимы от стека и среды.

- Здесь MUST находиться базовые инварианты, глоссарий и универсальные правила.
- Нормативные записи core MUST храниться модульно: один rule-entry на один файл в `aagf/docs/spec/core/rules/*.yaml`.
- Индекс модульных core-правил MUST храниться в `aagf/docs/spec/core/rules.index.yaml`.
