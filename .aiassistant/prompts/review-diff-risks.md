# 1. Review Diff With Risk Focus

## 1.1. Purpose

1.1.1. Review current diff and prioritize:
1.1.1.1. Functional regressions.  
1.1.1.2. Edge-case failures.  
1.1.1.3. Hidden coupling risks.  
1.1.1.4. Missing tests.  
1.1.1.5. Security/performance concerns.  

## 1.2. Response format

1.2.1. Findings by severity (high -> low).  
1.2.2. Evidence (file + line).  
1.2.3. Suggested fix per finding.  
1.2.4. Residual risk if unchanged.
